## üöÄ Quick Start Guide

### Option 1: View Locally (Instant)
1. Double-click `index.html` to open in your browser
2. That's it! Your portfolio is live locally

### Option 2: Local Development Server
1. Double-click `deploy.bat`
2. Choose option 2 (Python) or 3 (Node.js)
3. Visit http://localhost:8000 in your browser

### Option 3: Deploy to GitHub Pages (Free Hosting)
1. Create a GitHub account (if you don't have one)
2. Create a new repository named `portfolio`
3. Run these commands in the portfolio folder:
   ```bash
   git init
   git add .
   git commit -m "Initial portfolio commit"
   git branch -M main
   git remote add origin https://github.com/orielmalik/portfolio.git
   git push -u origin main
   ```
4. Go to repository Settings ‚Üí Pages
5. Select branch: `main`, folder: `/ (root)`
6. Your site will be live at: `https://orielmalik.github.io/portfolio/`

### Option 4: Deploy to Netlify (Easiest)
1. Go to https://app.netlify.com/drop
2. Drag the entire `portfolio` folder to the page
3. Your site is live instantly with a custom URL!
4. Optional: Configure custom domain
